# Cisco HSRP Lab

## Description

Ce projet présente une implémentation du protocole Cisco HSRP (Hot Standby Router Protocol) afin d'assurer la redondance de la passerelle par défaut.

## Topologie

- 2 Routeurs HSRP
- 1 Routeur LAN
- 2 Switches
- 3 PC
